## Physics Simulator

A simple newtonian physics simulator built in Rust. The main goal of the project was to learn the rust programming language. 
Particles are modelled as circles in 2D space, collision detection also works. The project also served as a good refresher of computer graphics. 

[Github Repository](https://github.com/nicknagi/rust-physics-simulation)
